We created this page to make voting for block producers quick and secure
Just follow these 5 simple steps:

1. Create a new folder on your PC/Mac
You'll need to save several files into this folder later

Scroll down for the next step

2. Download eos.js v14.1.1 and put to folder created in step 1
EOS.js is required for secure offline transaction generation. Make sure you are downloading the script from official EOS repository and not from any other source

Scroll down for the next step

3. Paste your account name into the input
Choose the amount of EOS to delegate (if available), press 'generate' and save generated HTML file into the folder created in step 1. The chosen amount will be delegated 50/50 to CPU and NET

Scroll down for the next step

4. Open downloaded 'secure-vote.html' file
Then fill in the form to vote. It's better turn internet connection off while you fill the form for better security. Once you fill the form, copy signed transaction.

Scroll down for the next step

5. Paste signed transaction to the form
The transaction we'll be broadcasted to EOS network. You need to turn internet connection back ON for this step.

Voting successful!
